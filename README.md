### 📘 梁慧星《民法总论》（第六版）

> *德国法学家创造了潘德克顿体系，梁慧星用四十年的时间，把它译成了中国民法学生听得懂的语言。*

梁慧星，1944 年生于四川眉山青神，中国社会科学院学部委员。从 1996 年第一版到 2021 年第六版，这本教材伴随了整整一代法律人从本科到执业的全过程。

第六版根据 2020 年《民法典》全面修订，全书十一章，382 千字：

| 章节 | 主题 | 民法学生的记忆点 |
|:---|:---|:---|
| 第一章 | 导论 | 从 ius civile 到《民法典》的千年之路 |
| 第二章 | 民事法律关系 | 权利主体、客体、内容的三位一体 |
| 第三章 | 自然人 | 出生到死亡，权利能力的起点与终点 |
| 第四章 | 法人·非法人组织 | 一群人如何变成法律上的"一个人" |
| 第五章 | 民事权利客体 | 物、行为、智力成果——权利指向什么 |
| 第六章 | **法律行为** | 全书的心脏：意思表示、效力瑕疵、条件与期限 |
| 第七章 | 代理 | 一个人替另一个人说话，法律认不认 |
| 第八章 | 诉讼时效 | 权利会过期，但不是所有权利都过期 |
| 第九章 | 期日·期间 | 法律时间如何计算 |
| 第十章 | 权利的行使 | 诚实信用、禁止权利滥用、私力救济 |
| 第十一章 | 民法的效力与解释 | 法律怎么用、怎么读、怎么解释 |

六次修订序言本身就是一部微缩的当代中国民法史：从 1996 年《合同法》尚在起草、到 2007 年《物权法》、2010 年《侵权责任法》、2017 年《民法总则》，直至 2021 年《民法典》全面落地。每一版序言的落款地点，从北京城南半夏书屋到云南昆明六榕居，像是在说：一个学者与他的国家，一起走了很远的路。

> 特别说明：梁慧星《民法总论（第六版）》PDF 因体积较大（约 209MB）尚未上传。本仓库收录基于该书的详细阅读笔记（`民法总论_梁慧星.md`）。

### 📗 朱庆育《民法总论》

> *如果梁慧星是"把民法讲清楚"，朱庆育就是"把民法的问题意识种进你的脑袋"。*

南京大学朱庆育教授的民法总论讲义，在华科以课堂笔记的形式被逐讲整理。每一讲分两个文件：**原版笔记**（课堂实录）和**讲义版**（结构化整理），覆盖民法总论所有核心板块。

| 板块 | 涉及的核心概念 |
|:---|:---|
| 法典编纂 | 潘德克顿体系、总则编的立法逻辑 |
| 权利理论 | 权利的本质、分类、行使与救济 |
| 权利主体 | 自然人、法人、非法人组织 |
| 法律行为的本质 | 意思表示的结构、法律行为的构成 |
| 法律行为的效力自治 | 私法自治、类型强制与类型自由 |
| 法律行为的类型 | 单方/双方/多方、要式/不要式 |
| 负担行为与处分行为 | 物权行为理论、分离原则与抽象原则 |
| 意思表示与解释 | 表示主义与意思主义的拉锯 |
| 法律行为效力瑕疵 | 无效、可撤销、效力待定的类型谱系 |
| 代理 | 显名主义、无权代理与表见代理 |

这份讲义的价值不在于"覆盖考点"，而在于它把德国民法教义学的思维方式，一行一行地写进了中文课堂。

### 🏫 喻家课件

华科课堂的原始 PPT，三个文件覆盖民法总论全部七章：

- `Chapter 1-2` 导论 + 民事法律关系
- `Chapter 3-4` 自然人 + 法人
- `Chapter 5-7` 民事权利客体 + 法律行为 + 代理

和喻家课的宏观经济学课件一样，架构清晰、直奔主题，适合课前预习和期末快速穿线。

### ✍️ 课堂讲稿 2026

侵权责任法的三份笔记：

| 文件 | 内容 |
|:---|:---|
| `民法：侵权责任.md` | 课堂笔记：归责原则、构成要件、特殊侵权 |
| `民法：侵权责任_讲稿.md` | 结构化讲稿：从法条到案例分析 |
| `民法：笔记0420.md` | 单次课随堂记录 |

侵权责任在《民法典》中独立成编（第七编），是中国民法典的特色创新之一。这三份笔记从课堂视角出发，把"过错""因果关系""损害赔偿"这些抽象概念还原到具体案件里。
